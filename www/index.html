Minima MiFi

Welcome to the Minma MiFi demo.

Here you will find some very simple demos that use the Minima MiFi Javscript library
which allow you to link a webpage to your phone running Minima.

DEFI Minima Style.

Please choose one of the following..

[1] Super Simple Block Explorer The most basic building block. Connect your phone to the webpage and receieve data.
[2] Basic Wallet Let's pick up the pace and allow you to send some Minima..
[3] Terminal The original bad boy - The Terminal
[4] Time Lock Funds Create a simple contract that locks funds away
[5] A Multi-Sig Contract An output that needs 2 people to spend
[6] Vault Wallet Hot wallet functionality with cold wallet security
[7] DEX Trade one token for another. Securely. Atomically
[8] Thunder Instant unlimited off-chain transactions.. Lightning by Minima..
[9] High Low Gambler 2 Player High low card game
[10] Poker Dice A full non-trivial state game
[11] Script IDE A very simple IDE for more complex Minima scripts
