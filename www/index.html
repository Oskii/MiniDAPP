Welcome to the Minima MiniDAPP portal!


Minima uses a unique, simple, yet extremely powerful system to allow Users to interact with the underlying blockchain and smart contracts. Embedded in the Minima application is a small webserver, and web-applications can be shared as .minidapp files ( basically a zipped up website ). These wrapped websites have access to the Minima command line, a SQL database, network sockets, file access - in fact everything you come to expect from any website you can create today.

A MiniDAPP is a standalone decentralised application that runs on the Minima network. MiniDAPPs can be unstoppable, uncensorable, resilient, decentralised applications that do not rely on anyone or anything other than a functioning Minima network. TRUE D.E.F.I.

Anyone can create and distribute a MiniDAPP.


What is a Smart Contract ?

Smart Contracts are programs that run on Minima.

They automatically manage money, tokens and value.

They can be hard to interact with directly.


What is a DAPP ?

DAPPs are Decentralised Applications.

They are front-ends to Smart Contracts.

They make interacting with Smart Contracts easy.


To be considered a TRUE DAPP..

1) No one can stop you using it

2) No one can censor who you use it with

3) No one can force a change to it

4) No one can turn it off


MINIDAPPS allow for this type of DAPP and more..
