Minima MiFi

Welcome to the Minima MiFi Demo Web Apps.

DEFI Minima Style.

Here you will find some simple demos that use the Minima MiFi Javscript library which allow you
to link a webpage to the Minima network via your phone or computer.

These are not complete sites. Just technical demos and examples of some of the things that are possible with Minima.

[1] Super Simple Block Explorer The most basic building block. Connect your phone to the webpage and receieve data.
[2] Basic Wallet Let's pick up the pace and allow you to send some Minima..
[3] Terminal The original bad boy - The Terminal
[4] Script IDE A basic IDE for Minima scripts. Endlessly Useful.
[5] Time Lock Funds Create a simple contract that locks funds away
[6] A Multi-Sig Contract An output that needs 2 people to spend. Import Export of partially signed transactions.
[7] Vault Wallet Hot wallet functionality with cold wallet security
[8] Shop You run a website. You sell stuff. Accept payment in Minima or other tokens.
[9] DEX Trade one token for another. Securely. Atomically
[10] Thunder Instant unlimited off-chain transactions.. Lightning 4 Minima
[11] Coin Flip Gambler 2 Player 50/50 gamble
[12] Poker Dice A non-trivial state game with complex logic.
